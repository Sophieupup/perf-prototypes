📸 周 / 月快照 · 锁定 · 解锁

Q11 用户指定 · admin 控制 freeze 时机

默认情况下系统按固定时间自动 freeze 快照（周快照周五 18:00 / 月快照月初 00:00）。锁定 = 默认时间不变；解锁 = admin 可临时改 freeze 时点。常用场景：节假日调整、经营会议提前、月底特殊处理。
